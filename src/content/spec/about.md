## 🌟 主题特性

### 🎨 设计与用户体验

- **现代优雅设计** - 简洁的极简界面和优美的排版
- **完全响应式** - 从手机到桌面的所有设备都进行了优化
- **深色/浅色模式** - 自动主题切换，过渡流畅
- **精美字体排版** - 使用 JetBrains Mono 字体增强可读性
- **平滑动画** - 流畅的页面过渡和交互效果

### 🔍 内容与搜索

- **高级搜索功能** - 由 [Pagefind](https://pagefind.app/) 提供支持，快速准确
- **增强 Markdown** - 扩展语法支持代码高亮和数学公式
- **交互式目录** - 长文章自动滚动导航
- **RSS 源生成** - 自动生成订阅源保持更新
- **阅读时间估计** - 显示文章预计阅读时长
- **文章分类功能** - 使用标签和分类组织内容

### 📱 特色页面

- **番剧追踪页面** - 追踪你的动画观看进度并评分
- **友情链接页面** - 展示友链网站的精美卡片
- **日记/动态页面** - 分享生活时刻，如社交媒体动态
- **归档页面** - 所有文章的有序时间线视图
- **关于页面** - 个性化自我介绍（当前页面！）

### 🛠 技术特性

- **增强代码块** - 由 [Expressive Code](https://expressive-code.com/) 提供支持
- **数学公式支持** - 使用 KaTeX 的 LaTeX 渲染
- **图片优化** - PhotoSwipe 相册和懒加载
- **SEO 优化** - 内置网站地图和 Meta 标签提升搜索可见性
- **性能优化** - 缓存优化实现快速加载
- **评论系统支持** - 集成 Twikoo 评论系统

### 🎯 高级 Markdown 功能

- **提示框** - 使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 创建美观的信息框
- **数学公式** - 使用 `$inline$` 和 `$$block$$` 语法编写 LaTeX 数学公式
- **GitHub 卡片** - 使用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 高级代码高亮支持行号
- **复制代码按钮** - 便捷的代码复制功能
